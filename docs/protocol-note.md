# Protocol Note

NuMetal-Q exposes a recursive proving lifecycle built around `seed`, `fuse`, `seal`, `resume`, and `verify`.

The benchmark and acceptance-demo executables exercise that lifecycle against the repository's canonical AG64 profile and bundled Metal artifacts. For the detailed algebraic and protocol background, see:

- [`MATH.md`](../MATH.md) for the consolidated math and protocol notes
- [`README.md`](../README.md) for the high-level package layout and public flow

This lightweight note exists so generated review bundles can point at a stable, repo-local protocol reference.
